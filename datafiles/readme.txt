
 ----- Kotonoha Run -----


マニュアル


1.概要
琴葉姉妹がエビフライやチョコミントアイスを食べながら走り続けてるラン&ジャンプ系ゲーム


2.あそびかた
キャラクター選択（タイトル画面）
・左矢印キー：琴葉 茜
・右矢印キー：琴葉 葵


基本操作（ゲーム画面）
・左矢印キー　：左に移動
・右矢印キー　：右に移動
・スペースキー：ジャンプ
エビフライやチョコミントアイスを食べるとスコアが増えます。
画面外に落ちてしまうとゲームオーバーになります。
